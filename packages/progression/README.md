# tonal-progression [![npm version](https://img.shields.io/npm/v/tonal-progression.svg?style=flat-square)](https://www.npmjs.com/package/tonal-progression)

[![tonal](https://img.shields.io/badge/tonal-progression-yellow.svg?style=flat-square)](https://www.npmjs.com/browse/keyword/tonal)

`tonal-progression` is a collection of functions to create, manipulate and analyse chord progressions.

This is part of [tonal](https://www.npmjs.com/package/tonal) music theory library.

You can install via npm: `npm i --save tonal-progression`

[Read the API documentation](http://danigb.github.io/tonal/api/module-progression.html).
